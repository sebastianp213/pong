# 🏓 Pong (Java)

Implementación del clásico juego Pong desarrollada en Java.

## 🎮 Descripción
Juego arcade donde el jugador controla una paleta y debe evitar que la pelota salga de su campo. Incluye lógica de colisiones y control en tiempo real.

## 🚀 Características
- Movimiento de paletas controlado por el usuario
- Detección de colisiones (pelota, paredes y paletas)
- Lógica básica de físicas
- Sistema de puntuación

## 🛠️ Tecnologías
- Java
- Programación Orientada a Objetos

## 👥 Autoría
Proyecto desarrollado de forma colaborativa.
